# Apps

`Onlystack` keeps runnable surfaces in `apps/`.

- `starter-api/` is the NestJS reference backend
- `starter-ios/` is the SwiftUI reference iOS app
- `starter-android/` is the Compose reference Android app
- `starter-web-public/` is the static-first public site starter
- `starter-web-admin/` is the operational admin starter
- `starter-web-app/` is the authenticated product web starter

Shared runtime code and contracts live in [libs/README.md](/Users/ted/dev/onlystack/libs/README.md) and [sentinel.yaml](/Users/ted/dev/onlystack/sentinel.yaml).
